Big O Question

Given an array of unsorted integers, create method called zero_sum that finds whether there is a slice (a smaller sequential array, within the initial array), that sums to 0 and returns it. It returns false otherwise.

example:
array = [1,4,3,27]
zero_sum(array) # returns false
array = [9,-3,2,1,6]
zero_sum(array) # returns [-3,2,1]
